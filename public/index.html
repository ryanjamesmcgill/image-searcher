FreeCodeCamp API Basejump: Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Usage search-example:

https://image-searcher-rjm.herokuapp.com/api/imagesearch/orions%20belt?offset=10
output
{
"url":"http://www.huddersfieldastronomy.org/sites/default/files/images/KielderNov2012%20044%20a.preview.JPG",
"snippet":"Orions Belt",
"thumbnail":"https://encrypted-tbn2.gstatic.com/images?q=tbn:ANd9GcSFyahzKSmFWMNLPuUySgpyHA6V7gomEcAYErzsPpE_9QXNC9ZJ6xTB3eZC",
"context":"http://www.huddersfieldastronomy.org/content/orions-belt"
}

Usage latest-queries:
